This mod was developed with the assistance of AI


The mod injects a controller into the game scene that monitors for a BoatMotor component. Once found, it overrides the internal weather variables (_weatherWindDirection and _weatherWindValue) in real-time. This ensures that even if the global world wind is blowing against you, the boat's sail "perceives" a perfect tailwind.

Creates a config-file that allows you to chage the maximum sail speed
